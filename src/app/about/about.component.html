¿Qué es "Mi Cocina"?

Este sitio fue creado con el propósito de aprender nuevas, pero ya bien establecidas, tecnologías de desarrollo de sitios web y también conocer e implementar servicios disponibles y gratuitos y su integración en aplicaciones Web modernas.

Utilizamos Angular, una plataforma de Google para construir aplicaciones web. Bootstrap en nuestro frontend nos ayuda a crear un diseño moderno y mobile oriented, y otros tantos componentes creados por la comunidad open source de desarrolladores y disponibles en el repositorio público de NPM nos ayudaron a crear "Mi Cocina".

Nuestro backend es un conjunto de REST web services desarrollados con Node.js y Express.js. Para almacenar los datos de nuestros usuarios y en función del tipo de información que vamos a almacenar, decidimos implementar Mongo DB, una base de datos escalable y orientada a documentos.

Nuestro código es open source, (no podía ser de otra forma), utilizamos un repositorio público de GitHub, el cual puede accederse aquí.

Hacemos a su vez un uso intensivo de diferentes servicios en la nube que son muy interesantes a la vez que generosos al ofrecernos una versión gratuita sin límite de tiempo.

Te invitamos a conocerlos e incorporarlos en tu próximo proyecto.

  • HEROKU: Serverless deployments en forma simple.
  • Auth0: Gestión de Autenticación y Autorización de usuarios completa ofreciendo también conectores a redes sociales.
  • Mongo DB Atlas: Nuestra Base de datos en la nube. ¡Incluyendo en la versión gratuita 500MB de espacio libre y un cluster!
  • Cloudinary: Manejo de imágenes y video en la nube y posibilidad de uso como CDN.
  • Trello: A traves de listas personalizables te permite fácilmente organizar las tareas de tu proyecto.
  • Testcollab: Herramienta online para administrar tus ciclos de testing.
  • Inkscape: Crea tus propios SVG!, El logo de este sitio fué creado con esta herramienta.
  • Sentry: Te ayuda a registrar los errores de tu aplicación en forma sencilla, categorizandolos y agrupandolos por sus ocurrencias. Se integra fácilmente con aplicaciones Node.JS y Angular.
  • Lucid chart: Crear diagramas de flujo, organigramas, prototipos de diseños web y más es posible con esta herramienta.
